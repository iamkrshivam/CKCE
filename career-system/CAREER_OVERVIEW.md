# Cybersecurity Careers – Overview

Cybersecurity offers a wide range of roles, from hands‑on technical positions to governance and management. A well‑structured career mapping helps learners understand prerequisite skills, domain knowledge, and progression paths.

## Why career mapping matters
- Provides clear learning goals.
- Aligns domain knowledge with job requirements.
- Enables skill gap analysis.
- Supports contribution‑based portfolio building.

## How domains connect to careers
Each career role requires expertise in one or more of the 14 CKCE domains. For example, a SOC Analyst needs Network Security, Security Operations, and Incident Response. A Cloud Security Engineer focuses on Cloud Security, IAM, and Application Security.

The file `DOMAIN_TO_CAREER_MAPPING.md` contains the detailed matrix.
