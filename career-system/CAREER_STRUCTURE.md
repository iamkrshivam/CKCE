# Career Structure Framework

All careers in CKCE follow a three‑level progression:

1. **Junior** – Basic domain knowledge, supervised tasks.
2. **Mid‑level** – Independent work, deeper specialisation.
3. **Senior/Lead** – Mentoring, architecture, strategy.

Additionally, each career has:
- **Core domains** (must have)
- **Secondary domains** (nice to have)
- **Recommended learning paths** (see `/learning-paths`)
- **Suggested projects** (see `/projects-hub`)
- **Typical certifications** (listed in `/resources-system`)

The actual skill definitions are maintained in `SKILL_GRAPH.md`.
