# Job Mapping – Domains to Roles

| Job Role | Core Domains | Secondary Domains |
|----------|--------------|--------------------|
| SOC Analyst | Security Operations (SOC), Network Security, Incident Response | Threat Intelligence, Vulnerability Management |
| Penetration Tester (defensive focus) | Application Security, Network Security, Vulnerability Management | Cryptography, Cloud Security |
| Cloud Security Engineer | Cloud Security, IAM, Network Security | Data Security, Application Security |
| Application Security Engineer | Application Security, Cryptography, IAM | Network Security, Vulnerability Management |
| Security Engineer | Network Security, Endpoint Security, IAM | Incident Response, Cloud Security |
| Malware Analyst (defensive) | Endpoint Security, Digital Forensics, Incident Response | Threat Intelligence, Cryptography |
| Incident Responder | Incident Response, Digital Forensics, SOC | Network Security, Threat Intelligence |
| Digital Forensics Analyst | Digital Forensics, Incident Response, Data Security | Endpoint Security, Cryptography |
| Threat Hunter | Threat Intelligence, SOC, Network Security | Incident Response, Vulnerability Management |
| IAM Engineer | IAM, Cloud Security, Cryptography | Application Security, GRC |
| GRC Analyst | GRC, Information Security, Data Security | IAM, Risk Management |
| Vulnerability Management Analyst | Vulnerability Management, Network Security, Application Security | Threat Intelligence, GRC |
| AI Security Researcher | Application Security, Cryptography, Research | Cloud Security, Threat Intelligence |

## Skill dependencies & career progression (text‑based)
- **Junior SOC Analyst** → **Mid SOC Analyst** → **Senior SOC Analyst** → **Threat Hunter** or **Incident Responder**.
- **Junior Penetration Tester** → **Mid Pen Tester** → **Senior Pen Tester** → **Application Security Engineer** or **Cloud Security Engineer**.
- **GRC Analyst** → **Compliance Manager** → **Security Director** (requires additional management skills).
- **IAM Engineer** → **IAM Architect** → **Identity & Access Manager**.

Cross‑domain skill building is encouraged: e.g., a Cloud Security Engineer benefits from IAM and Network Security.
