# Cloud Security Domain

What is it? – Securing cloud computing environments (IaaS, PaaS, SaaS).  
Why important? – Shared responsibility model creates new risks.  
Industry usage – CSPM, CWPP, IAM policies, container security.  
Contributions – Cloud security frameworks, serverless security concepts, Kubernetes hardening.  
Difficulty – Beginner (shared responsibility), Intermediate (IAM policies), Advanced (cloud forensics), Research (multi‑cloud posture management).  
Connections – Network Security, IAM, Application Security.  
Ethical – No real cloud credentials or live misconfiguration scans.
