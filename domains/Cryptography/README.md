# Cryptography Domain

What is it? – Mathematical techniques for secure communication.  
Why important? – Foundation of encryption, integrity, and non‑repudiation.  
Industry usage – TLS, disk encryption, digital signatures.  
Contributions – Algorithm explanations (AES, RSA, ECC), protocol descriptions, key management concepts.  
Difficulty – Beginner (symmetric vs asymmetric), Intermediate (hash functions), Advanced (PKI, side‑channel defences), Research (post‑quantum crypto).  
Connections – Data Security, Network Security, IAM.  
Ethical – No implementation of cryptographic primitives. No real private keys.
