# Endpoint Security Domain

What is it? – Securing laptops, servers, mobile devices.  
Why important? – Endpoints are the most common initial infection vector.  
Industry usage – Antivirus, EDR, application allow‑listing.  
Contributions – OS hardening guides, EDR architecture, secure boot principles.  
Difficulty – Beginner (malware types), Intermediate (process injection detection), Advanced (EDR bypasses and defences), Research (memory scanning techniques).  
Connections – SOC, Incident Response, Threat Intelligence.  
Ethical – No live malware creation. Only defensive detection methods.
