# Security Operations (SOC) Domain

What is it? – Centralised monitoring, detection, and response.  
Why important? – Continuous vigilance reduces dwell time.  
Industry usage – SIEM, SOAR, alert triage, threat hunting.  
Contributions – SOC processes, playbook examples, log analysis concepts.  
Difficulty – Beginner (log types), Intermediate (SIEM query writing), Advanced (threat hunting methodologies), Research (user behaviour analytics).  
Connections – Incident Response, Threat Intelligence, Network Security.  
Ethical – No real customer logs. Only synthetic or anonymised patterns.
