# Cloud Security – Scope

## Core topics
- Shared responsibility model
- Cloud service models (IaaS, PaaS, SaaS)
- Cloud deployment models (public, private, hybrid, multi‑cloud)
- Identity and access management in cloud (IAM roles, policies)
- Cloud network security (VPC, security groups, WAF)
- Data protection in cloud (encryption, key management)
- Cloud security posture management (CSPM)
- Container security (Docker, Kubernetes)
- Serverless security (AWS Lambda, Azure Functions)
- Cloud compliance frameworks (CIS benchmarks, SOC2)

## Relationship with other domains
- **IAM** – Cloud identity federation and privilege escalation.
- **Network Security** – Virtual networking, firewalls in cloud.
- **Application Security** – Secure DevOps (DevSecOps).
- **GRC** – Cloud‑specific compliance.

## Learning structure
- **Beginner** – Shared responsibility, basic IAM policies.
- **Intermediate** – VPC design, security groups, CSPM tools.
- **Advanced** – Kubernetes security, cloud forensics.
- **Research** – Multi‑cloud posture management, confidential computing.

No real cloud account credentials or live misconfiguration scans. Only theoretical descriptions.
