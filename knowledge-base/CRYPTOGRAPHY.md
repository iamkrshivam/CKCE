# Cryptography – Scope

## Core topics
- Symmetric encryption (AES, ChaCha20, block/stream ciphers)
- Asymmetric encryption (RSA, ECC, Diffie‑Hellman)
- Hash functions (SHA‑2, SHA‑3, BLAKE2)
- Message authentication codes (HMAC)
- Digital signatures and certificates
- Public Key Infrastructure (PKI) (CA, CRL, OCSP)
- Cryptographic protocols (TLS, IPsec, SSH)
- Key management (generation, storage, rotation, destruction)
- Quantum‑safe / post‑quantum cryptography

## Relationship with other domains
- **Data Security** – Encryption of data at rest and in transit.
- **Network Security** – TLS, IPsec, VPNs.
- **IAM** – Password hashing, certificate‑based authentication.
- **Application Security** – Secure random number generation, crypto APIs.

## Learning structure
- **Beginner** – Symmetric vs asymmetric, hash functions, basic TLS handshake.
- **Intermediate** – Block cipher modes, PKI, digital signatures.
- **Advanced** – Side‑channel defences, protocol analysis (TLS 1.3).
- **Research** – Post‑quantum algorithms, homomorphic encryption.

No cryptographic implementations (code). Only mathematical descriptions and usage guidelines. No actual private keys or random number generation seeds.
