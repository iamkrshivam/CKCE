# Data Security – Scope

## Core topics
- Data at rest, in transit, and in use protection
- Encryption (symmetric, asymmetric, hashing)
- Tokenisation and data masking
- Data Loss Prevention (DLP) strategies
- Database security (access controls, auditing, encryption)
- Data retention and destruction policies
- Privacy‑enhancing technologies (PETs)
- Data classification and labelling

## Relationship with other domains
- **Cryptography** – Encryption algorithms and key management.
- **Cloud Security** – Data stored in cloud services (S3, Azure Blob).
- **GRC** – Compliance with data protection laws (GDPR, CCPA).
- **Endpoint Security** – Data on laptops and mobile devices.

## Learning structure
- **Beginner** – Data classification, basic encryption concepts.
- **Intermediate** – DLP architecture, database auditing.
- **Advanced** – Tokenisation vs encryption, secure multiparty computation.
- **Research** – Homomorphic encryption, differential privacy.

No real customer or personal data may be referenced. Only synthetic examples.
