# Endpoint Security – Scope

## Core topics
- Operating system hardening (Windows, Linux, macOS)
- Antivirus / Anti‑malware (signature‑based, behaviour‑based)
- Endpoint Detection and Response (EDR)
- Application allow‑listing / deny‑listing
- Patch management
- Device control (USB, peripherals)
- Mobile device security (MDM, containerisation)
- Disk encryption (BitLocker, FileVault, LUKS)

## Relationship with other domains
- **Incident Response** – Endpoint investigation and containment.
- **Digital Forensics** – Acquisition of endpoint disk/memory images.
- **Network Security** – Endpoint‑to‑network communication policies.
- **Vulnerability Management** – Endpoint vulnerability scanning.

## Learning structure
- **Beginner** – OS hardening checklists, antivirus principles.
- **Intermediate** – EDR architecture, endpoint log analysis.
- **Advanced** – Memory protection bypasses and defences, rootkit detection.
- **Research** – Next‑gen AV, AI‑based endpoint threat detection.

Contributors must not include malware creation or live privilege escalation steps. Only defensive configurations and detection methods.
