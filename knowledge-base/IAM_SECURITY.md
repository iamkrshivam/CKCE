# Identity and Access Management (IAM) – Scope

## Core topics
- Identity lifecycle (provisioning, management, de‑provisioning)
- Authentication factors (something you know, have, are)
- Multi‑factor authentication (MFA) and adaptive authentication
- Authorisation models (RBAC, ABAC, ReBAC)
- Single Sign‑On (SSO) and federation (SAML, OAuth2, OpenID Connect)
- Privileged Access Management (PAM)
- Identity governance and administration (IGA)
- Directory services (LDAP, Active Directory)

## Relationship with other domains
- **Cloud Security** – IAM in AWS, Azure, GCP.
- **Application Security** – App‑level authentication flows.
- **Cryptography** – Certificates, password hashing.
- **GRC** – Access certification, segregation of duties.

## Learning structure
- **Beginner** – Password policies, MFA types, RBAC basics.
- **Intermediate** – OAuth2 and SAML flows, LDAP queries.
- **Advanced** – Privileged session management, identity federation.
- **Research** – Decentralised identity (DID), zero‑knowledge proofs.

No actual credentials, live directories, or configuration dumps. Only architectural descriptions.
