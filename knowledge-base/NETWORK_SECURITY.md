# Network Security – Scope

## Core topics
- OSI & TCP/IP models
- Network protocols (IPv4, IPv6, TCP, UDP, ICMP, DNS, HTTP, BGP)
- Firewalls (stateless, stateful, NGFW)
- IDS/IPS (signature‑based, anomaly‑based)
- VPNs (IPsec, WireGuard, OpenVPN)
- Network segmentation (VLANs, ACLs)
- Zero Trust Network Access (ZTNA)

## Relationship with other domains
- **Cloud Security** – Virtual networks, security groups.
- **Incident Response** – Network‑based containment and forensic log analysis.
- **Cryptography** – IPsec, TLS.
- **Threat Intelligence** – Network IoCs (IPs, domains).

## Learning structure
- **Beginner** – Subnetting, basic firewall rules, Wireshark basics.
- **Intermediate** – Dynamic routing, IDS rule writing, network troubleshooting.
- **Advanced** – Network forensics, BGP security, SDN security.
- **Research** – AI‑driven network defence, new transport protocols.

Contributors must not include offensive network attack techniques (e.g., ARP spoofing steps).
