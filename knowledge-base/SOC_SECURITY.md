# Security Operations (SOC) – Scope

## Core topics
- SOC tiers (Tier 1 triage, Tier 2 investigation, Tier 3 hunting)
- Security Information and Event Management (SIEM)
- Security Orchestration, Automation, and Response (SOAR)
- Log management and normalisation
- Alert triage and prioritisation
- Use case development (detection rules)
- Threat hunting methodologies
- SOC metrics (MTTD, MTTR)

## Relationship with other domains
- **Incident Response** – Escalation from SOC to IR team.
- **Threat Intelligence** – Enriching alerts with IoCs.
- **Network Security** – Analysing netflow, firewall logs.
- **Endpoint Security** – EDR alerts.

## Learning structure
- **Beginner** – Log types, SIEM dashboards, alert categories.
- **Intermediate** – Writing detection rules (pseudocode), playbooks.
- **Advanced** – SOAR automation, threat hunting tactics.
- **Research** – User and entity behaviour analytics (UEBA).

No real customer logs or live alert data. Only synthetic log formats.
