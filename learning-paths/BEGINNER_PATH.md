# Beginner Path – Foundation in Cybersecurity

**Target**: No prior cybersecurity knowledge.  
**Duration**: Approximately 6 months (self‑paced).  

## Phase 1: IT Fundamentals
- Domain: Information Security (Beginner)
- Topics: CIA triad, basic networking, OS concepts.

## Phase 2: Network & Endpoint Basics
- Domains: Network Security (Beginner), Endpoint Security (Beginner)
- Skills: Subnetting, firewall concepts, antivirus principles.

## Phase 3: Security Operations Introduction
- Domain: SOC (Beginner)
- Skills: Log analysis, SIEM overview.

## Phase 4: Choose a specialisation
- SOC Analyst, GRC, or IAM path (see respective path files).

**Capstone project**: Build a simple home lab with pfsense and analyse firewall logs.
