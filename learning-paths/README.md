# Learning Paths – Framework

Learning paths are sequential curricula that guide a learner from zero to job‑ready in a specific role. Each path references domain difficulty folders, skills from the skill graph, and suggested projects.

All paths are Markdown files that list:
- Target career role
- Prerequisite knowledge
- Phases (each with domain, difficulty level, and expected outcomes)
- Recommended resources (from `/resources-system`)
- Capstone project (from `/projects-hub`)

No actual content – only the structure and mapping rules.
