# SOC Analyst Learning Path

**Target career**: SOC Analyst (Tier 1 → Tier 2)  
**Prerequisites**: Basic IT literacy, networking fundamentals (OSI model, IP addresses).  
**Duration**: 6–9 months (self‑paced).

## Phase 1: Security Foundations
- **Domains**: Information Security (Beginner), Network Security (Beginner)
- **Skills**: CIA triad, common cyber threats, basic subnetting, TCP/UDP ports.
- **Resources**: `/resources-system` – introductory books and free online courses.

## Phase 2: Log Analysis and SIEM
- **Domains**: SOC (Beginner), Network Security (Intermediate)
- **Skills**: Log formats (syslog, Windows Event Log), SIEM query basics, alert triage.
- **Prerequisites**: Phase 1.

## Phase 3: Incident Detection
- **Domains**: SOC (Intermediate), Incident Response (Beginner), Threat Intelligence (Beginner)
- **Skills**: Correlation rules, IoC enrichment, phishing analysis.
- **Prerequisites**: Phase 2.

## Phase 4: Threat Hunting Fundamentals
- **Domains**: Threat Intelligence (Intermediate), Endpoint Security (Intermediate)
- **Skills**: MITRE ATT&CK tactics, hunting hypotheses, EDR basics.
- **Prerequisites**: Phase 3.

## Capstone Project (skeleton)
- **Project**: Build a mini SIEM dashboard using open‑source tools (conceptual description – no code).
- **Domains**: SOC, Network Security.
- **Outcome**: Alert triage playbook and sample dashboard layout.

> All skills and projects are defined at the framework level. Contributors will later add detailed content to the respective domain folders.
