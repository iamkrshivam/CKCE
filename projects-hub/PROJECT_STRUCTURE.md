# Project Structure Template (skeleton)

When you contribute a project description, follow this exact structure:

```markdown
# Project Title

## Domain(s)
List of CKCE domains.

## Difficulty
Beginner / Intermediate / Advanced / Research

## Careers
Relevant roles (from `career-system/JOB_MAPPING.md`).

## Description
2‑3 sentences explaining what the project does (defensive focus).

## Prerequisite Skills
- Skill 1 (from SKILL_GRAPH.md)
- Skill 2

## Expected Deliverables
- What the learner will produce (e.g., a Python script, a set of Snort rules).

## Resources Needed
- Tools (only names, no downloads)
- Datasets (synthetic or public)

## Evaluation Criteria
How the project can be checked for correctness.
