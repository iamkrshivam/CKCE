# Career Resources Guide – Classification Schema

Career resources help learners prepare for specific job roles. They include certification guides, interview preparation, and career roadmaps.

## Resource types
- **Certification guides** – Overview of certs (e.g., Security+, CISSP, OSCP – defensive use only).
- **Interview prep** – Common questions for each role.
- **Resume templates** – Generic, role‑specific templates.
- **Career roadmaps** – Text‑based progression charts.

## Metadata required
```markdown
- Career role: [from JOB_MAPPING.md]
- Resource type: (Certification / Interview / Resume / Roadmap)
- Difficulty: Beginner / Mid‑level / Senior
- Cost: Free / Paid
- Format: Article / Video / Template
Ethical note
Certifications that teach offensive skills must be described only from a defensive perspective (e.g., “understanding attacker tools to better defend”). No promotion of illegal activities.

All resources will be added by contributors under /resources-system/career/ in the future – this file only defines the schema.
