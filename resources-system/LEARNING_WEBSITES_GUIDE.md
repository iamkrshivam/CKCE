# Learning Websites Guide – Classification Schema

This file defines how to add references to external websites that offer cybersecurity education.

## Allowed website types
- **Interactive platforms** – TryHackMe, Hack The Box (only free/educational rooms with defensive focus).
- **Video tutorial sites** – YouTube channels, Coursera, edX (must have free tiers).
- **Documentation hubs** – OWASP, NIST, Microsoft Learn.
- **Blogs** – Security company blogs (e.g., SANS, Mandiant) – no personal blogs with offensive content.

## Metadata required
```markdown
- Website name: [Name]
- Primary domain(s): [list]
- Difficulty coverage: Beginner / Intermediate / Advanced
- Cost: Free / Freemium / Paid subscription
- Defensive focus: Yes/No (must be Yes)
- Description: (one sentence)
```
Restrictions
No websites that primarily host exploit code or attack tutorials.

No forums with real‑world hacking discussions (e.g., RaidForums).

No URL shorteners – use direct, stable links (when added later).
